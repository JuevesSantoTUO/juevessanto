# 🎭 Jueves Santo - Tuna Universitaria de Oviedo

Aplicación Web Progresiva (PWA) diseñada para la gestión y consulta de información durante la festividad del Jueves Santo. Una herramienta moderna que combina la tradición de la Tuna con la tecnología actual.

## 🚀 Características principales
- **Cancionero Digital:** Acceso rápido a las letras y acordes de la Tuna.
- **Itinerarios:** Rutas detalladas y horarios de las procesiones y rondas.
- **Galería Multimedia:** Archivo fotográfico y de vídeo con navegación táctil.
- **Histórico de Asistentes:** Registro migrado y optimizado de los participantes.
- **PWA Ready:** Instalable en dispositivos móviles para uso offline.

## 📱 Instalación
Para instalar la App en tu smartphone:
1. Abre el enlace en Chrome (Android) o Safari (iOS).
2. Selecciona **"Añadir a la pantalla de inicio"** o **"Instalar aplicación"**.
3. El icono de la Tuna aparecerá en tu menú de aplicaciones.

## 🛠️ Tecnologías utilizadas
- HTML5 / CSS3 (Diseño "Negro y Oro" v1.8)
- JavaScript Vanilla
- Service Workers para soporte offline
- Google Forms para el sistema de participación
